Zencherry JavaServer Pages Content Management System

Installation: Famous 5-minute install

Unzip the package in an empty directory and upload everything.

Import the zcms.sql to your MySQL Database.

Go to config.jsp and configure your Data Base username, Data Base password, Data Base name, Site Title, and others in the file.


Open http://www.yoursite.com/?dir=admin in your browser. 

Login as USERNAME: admin, PASSWORD: admin

Change the admin password inside your Data Base under "xxx_users"
